# 物流管理系统 - 布局调整说明

## 调整内容
根据需求，将所有管理页面的正文内容从居中布局调整为靠上对齐布局。

## 具体变更
- 修改了所有页面的 `.main-content` 样式
- 添加了右边距以保持适当的空间
- 调整了宽度计算以适应侧边栏
- 设置了最小高度以充分利用视口空间

## 影响的页面
- 首页 (index.html)
- 订单管理 (orders.html)
- 租户管理 (tenants.html)
- 入驻申请管理 (application-list.html)

## 布局效果
- 内容紧贴顶部和左侧（避开侧边栏）
- 右侧保留适当边距
- 底部延伸至窗口底部
- 更符合管理后台的使用习惯
