# Admin 平台导航栏统一修复记录

## 日期
2026 年 2 月 19 日

## 问题描述
用户反馈：【地图服务管理】菜单在平台管理员页面中不一致，点击某些页面（如控制台）会出现，点击其他页面（如订单管理）就会消失。

## 问题原因
所有平台管理员（admin-web）页面的导航栏中**都缺少三个地图相关菜单项**（除了 dashboard.html、map-management.html、vehicle-tracking.html）。

## 修复内容

### 统一导航菜单结构
在所有平台管理员页面的侧边栏导航中，统一添加三个地图相关菜单项，位置在"车型库维护"之后。

### 添加的三个菜单项

```html
<li><a href="/map-management.html"><i class="fas fa-map-marked-alt"></i> <span>地图服务管理</span></a></li>
<li><a href="/stop-points.html"><i class="fas fa-map-marker-alt"></i> <span>停靠点管理</span></a></li>
<li><a href="/vehicle-tracking.html"><i class="fas fa-location-arrow"></i> <span>车辆位置追踪</span></a></li>
```

## 验证步骤

1. 访问 http://localhost:3000/admin-web/login.html 登录
2. 访问 http://localhost:3000/admin-web/dashboard.html
3. 依次点击每个菜单项
4. 确认【地图服务管理】、【停靠点管理】、【车辆位置追踪】在所有页面都可见
5. 确认导航栏样式保持一致
