# 承运商页面导航栏统一修复记录

## 日期
2026 年 2 月 19 日

## 问题描述
用户反馈：【停靠点管理】菜单与其他导航栏不一致，点击控制台会出现，点击订单管理就会消失。

## 问题原因
所有承运商页面的导航栏中**都缺少"停靠点管理"菜单项**（除了 carrier-stop-points.html 本身）。

## 修复内容

### 统一导航菜单结构
在所有承运商页面的侧边栏导航中，统一添加"停靠点管理"菜单项，位置在"申请资料"和"地图服务"之间。

| 文件 | 修复内容 |
|------|---------|
| `carrier-stop-points.html` | 统一 Logo 文字、HTML 标签 |
| `orders.html` | 添加"停靠点管理"菜单 |
| `vehicles.html` | 添加"停靠点管理"菜单 |
| `quotes.html` | 添加"停靠点管理"菜单 |
| `applications.html` | 添加"停靠点管理"菜单 |
| `profile.html` | 添加"停靠点管理"菜单 |

### 统一的导航菜单结构

```
承运商系统 (侧边栏 Logo)
├── 控制台 (carrier-dashboard.html)
├── 订单管理 (orders.html)
├── 报价管理 (quotes.html)
├── 车辆管理 (vehicles.html)

```
