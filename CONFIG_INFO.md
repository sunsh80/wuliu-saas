# 物流项目配置说明

## .env 文件配置项

当前 `.env` 文件包含以下重要配置：

### 会话配置
- `SESSION_SECRET` - 用于加密会话cookie的密钥
  - 重要性：安全关键，防止会话劫持
  - 建议：在生产环境中使用强随机密钥

### 数据库配置
- `DATABASE_PATH` - SQLite数据库文件路径
  - 重要性：指定数据存储位置
  - 建议：确保路径存在且有写入权限

### 服务器配置
- `PORT` - 服务器监听端口
  - 重要性：指定服务访问端口
  - 建议：根据部署环境调整端口

## 已移除的配置
- `REDIS_URL` - 已移除，改为内存会话存储

## 安全建议
1. 不要在版本控制中提交真实的 .env 文件
2. 在生产环境中使用强密码和密钥
3. 定期轮换密钥
