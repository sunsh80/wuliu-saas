# 流程图文档

此目录包含项目的各种流程图和结构文档，用于帮助理解和维护系统。

## 文件列表

- `OrderFlow.mmd` - 订单流转流程图（Mermaid格式）
- `PROJECT_STRUCTURE.md` - 项目结构说明
- `PROJECT_SUMMARY.md` - 项目总结报告
- `project-structure.md` - 项目结构文档

## 使用说明

- `OrderFlow.mmd` 文件可在支持 Mermaid 语法的编辑器或在线工具中查看
- 所有流程图均反映了当前系统的实际业务流程
- 如需更新流程，请同步修改相关文档和代码注释
