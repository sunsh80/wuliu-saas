```
     │                          │                          │
     │                          ├── 2. 查看订单 ──────────>│ 监管
     │                          │                          │
     │                          ├── 3. 报价 ──────────────>│ 备案
     │                          │                          │
     ├── 4. 选择报价 <──────────│                          │
     │                          │                          │
     │<──────── 5. 支付（混合支付）─────────────────────────│
     │                          │                          │
     │                          ├── 6. 接单配送 ──────────>│ 追踪
     │                          │                          │
     ├── 7. 确认收货 <──────────│                          │
     │                          │                          │
     ├── 8. 评价 <──────────────┼────────── 9. 回复 ──────>│ 审核
     │                          │                          │
```
